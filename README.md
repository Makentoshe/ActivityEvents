# ActivityEvents

Events List: 
- onCreate
- onPostCreate
- onStart
- onResume
- onPostResume
- onPause
- onStop
- onDestroy
- onAttachedToWindow
- onDetachedFromWindow
- onSaveInstanceState
- onRestoreInstanceState
- onBackPressed

The most part of the events calls on device rotation or application open/close. onBackPressed event calls on the back button click.
